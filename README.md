# Data Minig Tool
Data Minig Tool Kit

## How to Run
The project is based on gradle build tool with java 10

In a command line run the following commands :
```
git clone https://github.com/mohammedi-haroune/data-minig-toolkit.git
cd data-minig-toolkit
./gradlew run
```

**Enjoy !**
